# New Generation Farms - Grant Strategy

## Overview

This document outlines grant opportunities for New Generation Farms and AgTools, prioritized by fit, funding amount, and timeline.

---

## Priority 1: USDA SBIR/STTR (Best Fit)

**Why this is #1:** AgTools is a technology product with commercial potential - exactly what SBIR funds.

### Program Details

| Aspect | Details |
|--------|---------|
| **Agency** | USDA NIFA |
| **Phase I Funding** | $125,000 - $175,000 |
| **Phase II Funding** | Up to $650,000 |
| **Duration** | 8 months (Phase I), 24 months (Phase II) |
| **Cost Share** | None required |
| **Website** | [NIFA SBIR/STTR](https://www.nifa.usda.gov/grants/programs/sbir-sttr) |

### Relevant Topic Areas

1. **8.2 Plant Production and Protection - Biology**
   - Pest/disease identification systems
   - AI-based crop health monitoring

2. **8.3 Plant Production and Protection - Engineering**
   - Precision agriculture technologies
   - Decision support systems

3. **8.5 Air, Water, and Soil** (if framing around sustainability)
   - Environmental monitoring
   - Input reduction technologies

4. **8.9 Small and Mid-Size Farms**
   - Technologies that improve farm profitability
   - Decision support for smaller operations

### AgTools Alignment

| SBIR Requirement | AgTools Feature |
|------------------|-----------------|
| Innovation | AI-powered pest/disease ID with 46+ species knowledge base |
| Commercial Potential | SaaS model, $199/month pricing planned |
| Technical Merit | ML/AI integration, GDD tracking, yield prediction |
| Societal Benefit | Reduces pesticide overuse, improves sustainability |

### Key Dates (Typical)

- **Phase I RFA Release:** September
- **Phase I Deadline:** January
- **Phase II Deadline:** September (for Phase I awardees)

### Action Items

- [ ] Register on Grants.gov
- [ ] Get DUNS/SAM registration current
- [ ] Monitor for FY2025/2026 RFA release
- [ ] Prepare 15-page technical proposal
- [ ] Identify letter of support partners (farmers, co-ops)

## Priority 2: Southern SARE Producer Grant

**Why this fits:** Funds farmer-led research - pairs perfectly with AgTools research tools.

### Program Details

| Aspect | Details |
|--------|---------|
| **Region** | Southern (includes Louisiana) |
| **Funding** | $10,000 - $30,000 typical |
| **Duration** | 1-2 years |
| **Eligibility** | Farmers and ranchers |
| **Website** | [Southern SARE Grants](https://southern.sare.org/grants/) |

### Project Ideas

1. **Sustainability Metrics Validation Study**
   - Use AgTools to track inputs and carbon footprint
   - Compare conventional vs. sustainable practices
   - Document cost savings and environmental benefits

2. **GDD-Based Pest Management Trial**
   - Use climate tracking to optimize spray timing
   - Measure input reduction vs. control fields
   - Calculate economic benefit

3. **AI Pest Identification Accuracy Study**
   - Field validation of AgTools AI identification
   - Compare to expert diagnosis
   - Document time savings

### Key Dates

- **Call for Proposals:** February
- **Deadline:** May
- **Awards Announced:** August

### Action Items

- [ ] Sign up for Southern SARE newsletter
- [ ] Identify research question for 2025 cycle
- [ ] Connect with Louisiana sustainable ag coordinator
- [ ] Prepare project narrative
